# Cambios para la app Taller Solis

Usa este archivo para anotar cambios que quieres pedir despues.

## Como pedir un cambio

Copia este formato y llena lo que necesitas:

```txt
Fecha:
Modulo:
Que quiero cambiar:
Como debe verse o funcionar:
Texto exacto que debe aparecer:
Que NO se debe tocar:
Prioridad:
```

## Ejemplos

```txt
Fecha: 16/06/2026
Modulo: Cotizacion inteligente
Que quiero cambiar: Que la IA pregunte primero por el nombre del cliente si falta.
Como debe verse o funcionar: Si falta cliente, debe decir "Leonardo, me falta el nombre del cliente".
Texto exacto que debe aparecer: Leonardo, me falta el nombre del cliente.
Que NO se debe tocar: No cambiar PDF ni historial.
Prioridad: Alta
```

```txt
Fecha: 16/06/2026
Modulo: PDF
Que quiero cambiar: Agregar telefono del taller al PDF.
Como debe verse o funcionar: Debe aparecer abajo del nombre TALLER SOLIS.
Texto exacto que debe aparecer: Tel. 55 1234 5678
Que NO se debe tocar: No cambiar totales ni conceptos.
Prioridad: Media
```
