HUELLA / BIOMETRIA

La huella funciona en celular cuando la app esta publicada en HTTPS, por ejemplo en GitHub Pages.
No funciona desde archivo local ni desde algunos navegadores internos como WhatsApp/Facebook. Abre el link en Chrome y despues instala la app en pantalla principal.

Primer uso:
1. Escribe usuario y contrasena.
2. Toca ACTIVAR HUELLA si aparece.
3. El celular pedira huella, rostro o PIN para crear la passkey.
4. Despues ya podras entrar con ENTRAR CON HUELLA en ese mismo celular.

Si abres el archivo directo desde la computadora, la huella puede no funcionar. Debe abrirse desde el link publicado de GitHub Pages.

Para usarla en el celular:
1. Abre el archivo en Chrome o Edge desde la computadora.
2. Tambien puedes copiar la carpeta web-pwa al celular y abrir index.html.
3. En Chrome puedes usar "Agregar a pantalla principal" para tenerla como app.
